# Índice de Skills del Proyecto

| Skill | Dominio | Cuándo usar |
|-------|---------|-------------|
| flutter-screen-implementation | flutter-ui | Crear o modificar pantallas y widgets de la app. |
| firebase-sync-repository | sync-cloud-local | Cambiar sincronización entre local y Firestore en repositorios. |
| android-methodchannel-alarms | android-native-bridge | Ajustar payloads o handlers entre Dart y Kotlin para alarmas. |
| medication-habit-scheduling | scheduling-domain | Modificar lógica de recurrencia para hábitos y medicación. |
| dart-analysis-and-testing | quality-gates | Validar cambios con `dart analyze` y pruebas según impacto. |
| documentation-txt-maintenance | technical-documentation | Actualizar `.txt` técnicos acorde a cambios implementados. |

## Cómo usar estas skills
1. Copia la skill relevante al contexto de tu sesión con el asistente.
2. O referencia el archivo de skill con: "Aplica la skill [nombre]".
3. El asistente seguirá el proceso definido automáticamente.
